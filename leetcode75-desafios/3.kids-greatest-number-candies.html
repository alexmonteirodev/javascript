Crianças com o maior número de doces

Não há crianças com doces. Você recebe uma array inteira candies, onde cada candies[i] representa o número de doces que a ith criança possui, e um número inteiro extraCandies, denotando o número de doces extras que você possui.

Retorna um resultado de array booleano de comprimento n, onde result[i] é verdadeiro se, depois de dar ao ith filho todos os doces extras, ele terá o maior número de doces entre todas as crianças, ou falso caso contrário.

Observe que várias crianças podem ter o maior número de doces.

Exemplo:

Input: word1 = "abc", word2 = "pqr"
Output: "apbqcr"
Explanation: The merged string will be merged as so:
word1: a b c
word2: p q r
merged: a p b q c r
