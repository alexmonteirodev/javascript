Máximo divisor comum de strings

Dado duas strings str1 e str2 , o problema nos pede para encontrar a maior string que pode ser repetida para compor ambas as strings originais. Ou seja, queremos encontrar uma string x tal que:

1. x seja uma “divisora” de str1 e str2.

2. A string x pode ser repetida múltiplas vezes para formar tanto str1 quanto str2.

Exemplo 1:

Entrada: str1 = "ABCABC", str2 = "ABC"

Saída: "ABC"
